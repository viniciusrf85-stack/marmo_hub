# OLX Pedra - Marketplace de Granito e Mármore

Sistema completo para comercialização de pedras ornamentais (granito e mármore) do Espírito Santo.

## 🎯 Características Principais

- **Sistema de Planos**: Empresas podem contratar planos de 1 a 21 anúncios (incrementos de 3)
- **Galeria de Fotos**: Cada material possui 1 foto principal + até 3 fotos adicionais
- **Multi-usuário**: 4 tipos de perfis (Administrador, Empresa, Cliente, Agenciador)
- **Busca Avançada**: Filtros por empresa, tipo de material, preço, etc.
- **Interface Moderna**: Inspirada em OLX, Mercado Livre e Webmotors

## 👥 Tipos de Usuários

### Administrador
- Gerenciar todas as empresas e usuários
- Aprovar/desaprovar anúncios
- Relatórios completos do sistema

### Empresa
- Cadastrar e editar dados da empresa
- Gerenciar materiais (pedras) e fotos
- Controlar preços e disponibilidade
- Visualizar estatísticas de visualizações

### Cliente
- Navegar pelo marketplace
- Buscar materiais com filtros avançados
- Visualizar detalhes e fotos dos materiais
- Entrar em contato com empresas

### Agenciador
- Intermediar negociações
- Cadastrar empresas parceiras
- Comissões sobre vendas

## 🚀 Tecnologias

- **Backend**: Node.js + Express + MySQL
- **Frontend**: React + Vite
- **Autenticação**: JWT
- **Upload de Arquivos**: Multer
- **Estilização**: CSS moderno + Responsivo

## 📦 Estrutura do Projeto

```
olx_pedra/
├── backend/          # API REST em Node.js
├── frontend/         # Interface React
├── database/         # Scripts SQL
├── uploads/          # Armazenamento de imagens
└── docs/            # Documentação
```

## 🛠️ Instalação Rápida

1. Instalar dependências:
```bash
instalar.bat
```

2. Configurar banco de dados (MySQL)
3. Iniciar o sistema:
```bash
iniciar.bat
```
